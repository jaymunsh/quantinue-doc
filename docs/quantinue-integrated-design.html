Quantinue
Quantinue · 2차 (MVP-2) · v4.0

AI가 제안하고, AI가 반박하고,
코드가 막는다.

공시·뉴스·기술지표·매크로를 읽어 AI가 매수를 제안하면, 또 다른 AI가 적대적으로 반박하고, 결정론적 코드가 리스크를 계산해 페이퍼 계좌로 집행한 뒤 5영업일 후 복기한다. 2차는 이 사이클을 사람 손 없이 자동으로 돌리고, 관리자·유저 화면을 얹어 AI 페이퍼 펀드 서비스로 완성한다. LLM이 검증 없이 돈에 닿는 경로는 0개.

파이프라인11단계 투자 성향공격형·안전형 계좌페이퍼·다계좌 운영무인 자동 LLMOpenAI
Status

현재 상태 — 어디까지 왔고 무엇을 만드나

1차 MVP(반자동 단일 파이프라인)는 코드로 완성됐다. 2차는 그 위에 자동 운영·다계좌·서비스 화면을 얹는다.

Overview

전체 흐름 한눈에

리서치(01~08)는 사이클당 1회 공용 실행, 계좌(09~10)는 성향별로 구독·실행, 11이 복기해 다음 판단으로 되돌린다.

01→11 파이프라인 다이어그램 + 리서치 공용/계좌 구독 구조 + 서비스 2면(관리자·유저) 도식
Schedule

시간·스케줄 — 무엇이 언제 도나

앱 내장 스케줄러가 역할별 실행 창(뉴욕 시간)에 슬롯 정렬 cycle_ts로 구동. 서머타임·휴장일 보정.

역할별 실행 창 표 + 하루 타임라인(KST 변환) + cycle_ts 슬롯 멱등 설명
Data model

데이터 모델 — ERD·스키마

원본 원장(1건=1행)과 집계 신호(종목당 사이클 1행)를 분리. 컨벤션 8규칙이 어휘의 정본.

ERD(Mermaid) + 테이블 총람 + 컨벤션 8규칙(규칙4 개정 반영) + MVP-2 신규 테이블(tb_user·tb_llm_usage·tb_benchmark_price 등)
