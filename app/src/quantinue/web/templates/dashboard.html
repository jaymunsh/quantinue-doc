투자 판단 전 과정을 검증하는 1차 MVP

Quantinue 1차 MVP는 시장 데이터를 수집해 투자 후보를 선별하고, 근거 분석·재검토·위험 계산을 거쳐 모의 주문 결과를 기록하는 전체 투자 판단 흐름을 검증하는 시스템입니다.

개발 목적수익률을 증명하거나 완전 자동매매를 운영하는 것이 아니라, 수집한 근거가 판단·주문·저장까지 끊기지 않고 추적되는지 확인하는 것입니다.

1차 기준 구성실제 공개 데이터 + PostgreSQL 저장소 + 로컬 모의계좌입니다. 현재 실행 상태는 바로 위의 실행 환경 배지가 기준 구성과 같은지 보여줍니다.

  1. 입력시장·기업의 공개 정보

    NASDAQ 상장사 후보, 시장 가격, FRED 미국 경제지표, SEC 미국 기업 공시, Google News 뉴스 목록을 수집합니다.

  2. 선별50 → 20 → 20

    50개 기업을 만들고 기술 지표로 20개를 분석한 뒤, 상세 판단 후보 20개를 확정합니다.

  3. 판단근거 종합과 반증
    전략가→비평가→리스크

    공시·뉴스·기술 신호를 종합하고, 누락·충돌을 반증한 뒤 위험 한도와 수량을 다시 계산합니다.

  4. 결과모의 체결과 추적 기록

    리스크 한도로 수량을 계산하고, 주문·체결과 데이터 출처의 연결 기록을 관계형 데이터베이스에 보존합니다.

1차의 의도적 제한

AI와 운영 자동화는 최소화

  • AI를 핵심 엔진으로 전면 활용하지 않음
  • 1차 시연은 로컬 LLM을 정해진 항목의 판단에만 제한적으로 사용
  • 뉴스는 전체 선별 후 대표 1건만 정밀 분석
  • 11단계는 리뷰 대상 등록이며 T+1~T+5 자동 리뷰는 제외

1차 MVP의 완료 기준
